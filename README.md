# calculadora-shell
Projeto de calculadora simples desenvolvida em Shell Script (Bash), executada em ambiente Linux. Permite realizar operações matemáticas básicas como soma, subtração, multiplicação e divisão via terminal, além de demonstrar criação de scripts e gerenciamento de permissões no sistema.

# Calculadora em Shell Script

Este repositório contém uma calculadora simples executada via terminal, desenvolvida em Shell Script (Bash).

O projeto foi criado como atividade prática, com o objetivo de aplicar conceitos de lógica de programação, uso de terminal Linux e controle de versões com Git.

## Funcionalidades
- Entrada de dois números pelo usuário
- Escolha da operação matemática
- Operações disponíveis:
  - Soma
  - Subtração
  - Multiplicação
  - Divisão
- Validação básica (ex: divisão por zero)

## Requisitos
- Sistema operacional Linux, macOS ou Windows com WSL
- Terminal para execução de scripts Bash

## Como executar

1. Acesse a pasta do projeto pelo terminal: cd caminho/do/projeto
2. Conceda permissão de execução: chmod +x calculadora.sh
3. Execute o script: ./calculadora.sh


## 📌 Observação
Este projeto também demonstra conceitos básicos de:
- Shell Script (Bash)
- Manipulação de entrada de dados
- Estruturas condicionais
- Permissões de arquivos no Linux
